未来一段时间的学习重点

摸索概率和统计的思维

学习 = 积累+再造

学习的动作 = 读书，看视频，做练习

学习的目的 = 理解，内化，直觉

学习的过程 = 通过迭代，度过一些列的阶段，如 “天书，似懂非懂，OK， insightful understanding, great intuition”

迭代 = 不断学习累计新知识 + 不断回顾，消化和更新学习过的内容

难点 = 迭代

学了新的，忘了旧的

想回顾旧知识，找不到在哪里

学的多了，知识太零碎，不系统，杂乱无章

无法迭代，无法再造

解决方案

将旧知识收集整理在手头，随时快速搜索查阅

查漏补缺，促进消化理解，纠正和深化认知

如何查漏补缺

和大家一起来做提问与回答（持续更新）

持续收集整理汇总这些问答，方便快速搜索查阅

在查漏补缺的过程中，梳理汇总所有的概念和知识点的迭代

如何问答并持续更新

使用github issues

这里拥有提问回答的所有必备工具

issues 的管理也比较方便

问答要求

好的问题，比回答更有价值！

好的问题，至少应该具备努力和认真的样子

  • 视频链接，时间定位
  • 哪里懂了，哪里不懂，迷惑点是什么
  • 提供符号和公式表达（如果问题需要）

认真的提问，必然会有认真的回答

P(隐含真实问题= X | 笼统的提问）< 10%

  • X can be any of mutliple questions, each prob is below 10%
  • no one can be certain what to think and how to answer
  • P(隐含真实问题 = a | 笼统的提问, 认真描述）> 80%
  • it is possible to answer
